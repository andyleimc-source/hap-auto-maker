# 视图类型注册中心

## 视图类型清单 (11 种, viewType 0-10)

### 基础视图 (0-5)
| viewType | 名称 | 验证 | 字段约束 | 二次保存 |
|----------|------|------|---------|---------|
| 0 | 表格视图 | ✓ | — | 分组: groupView |
| 1 | 看板视图 | ✓ | type=11 单选 | viewControl |
| 2 | 层级视图 | ✓ | type=29 自关联 | childType + layersControlId |
| 3 | 画廊视图 | - | — | coverCid |
| 4 | 日历视图 | - | type=15/16 日期 | calendarcids |
| 5 | 甘特图 | ✓ | type=15/16 x2 | begindate + enddate |

### 高级视图 (6-9)
| viewType | 名称 | 验证 | 字段约束 | 说明 |
|----------|------|------|---------|------|
| 6 | 详情视图 | - | — | 单条记录表单 |
| 7 | 地图视图 | - | type=24/40 地理 | 需地区或定位字段 |
| 8 | 快速视图 | - | — | 打印/快速预览 |
| 9 | 资源视图 | - | type=26+日期 | 按成员的时间线 |

### 插件 (10)
| viewType | 名称 | 说明 |
|----------|------|------|
| 10 | 自定义视图 | 需安装插件 |
