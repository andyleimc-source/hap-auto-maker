# 工作流触发 - 按日期字段触发 API

> 接口：`POST /api/workflow/saveNode`（触发节点）
> 来源：浏览器抓包（5次保存），辅助帮助文档验证
> 认证：Web (Cookie + Authorization)

---

## 与工作表事件触发的关键区别

| 字段 | 工作表事件触发 | 按日期字段触发 |
|------|-------------|-------------|
| `appType` | `1` | `6` |
| `name` | `"工作表事件触发"` | `"按日期字段触发"` |
| `assignFieldId` | ❌ 无此字段 | ✅ 监听的日期字段 ID |
| `executeTimeType` | ❌ 无此字段 | ✅ 触发时机偏移类型 |
| `number` | ❌ 无此字段 | ✅ 偏移数量 |
| `unit` | ❌ 无此字段 | ✅ 偏移单位 |
| `endTime` | ❌ 无此字段 | ✅ 当天执行时刻 |
| `executeEndTime` | ❌ 无此字段 | ✅ 结束执行时间字段引用 |
| `frequency` | ❌ 无此字段 | ✅ 重复周期 |
| `triggerId` | `"1"~"4"` 触发时机 | `"2"` 新增或更新（验证中）|

## 参数说明

### 核心参数

| 参数 | 类型 | 说明 |
|------|------|------|
| appType | integer | 固定 `6` |
| name | string | 固定 `"按日期字段触发"` |
| assignFieldId | string | 监听的日期字段 ID（系统字段如 `ctime`=创建时间）|
| assignFieldIds | array | 固定空数组 `[]`（此触发类型不用）|
| triggerId | string | 固定 `"2"`，按日期字段触发无 UI 可配置，意为"新增记录或修改日期字段时均纳入监听" |

### 触发时机（executeTimeType）

| executeTimeType | 含义 | number | endTime |
|----------------|------|--------|---------|
| `0` | 当天指定时刻触发 | `0` | `"08:00"` 等具体时间 |
| `1` | 日期前 N 单位触发 | 偏移量 | 可选具体时刻 |
| `2` | 日期后 N 单位触发 | 偏移量 | `""` |

### 偏移单位（unit）✅ 已全部验证

| unit | 含义 |
|------|------|
| `1` | 分钟 |
| `2` | 小时 |
| `3` | 天 |

### 重复周期（frequency）✅ 已全部验证

| frequency | 含义 |
|-----------|------|
| `0` | 不重复 |
| `1` | 每年 |
| `2` | 每月 |
| `3` | 每周 |

### 时间字段

| 参数 | 类型 | 说明 |
|------|------|------|
| endTime | string | 当天执行的具体时刻，如 `"08:00"`，当 executeTimeType=2 时为 `""` |
| executeEndTime | string | 结束执行时间的字段引用（如 `"wfdtime"` = 剩余时间字段），无则为 `""` |
| time | string | 目前抓包均为 `""`，待确认用途 |

## 5次抓包对比

| 次 | executeTimeType | number | unit | endTime | frequency | 语义推断 |
|----|----------------|--------|------|---------|-----------|---------|
| 1 | `2` | `15` | `3` | `""` | `1` | 日期后15天触发，每年重复 |
| 2 | `0` | `0` | `3` | `"08:00"` | `1` | 当天08:00触发，每年重复 |
| 3 | `1` | `15` | `1` | `"08:00"` | `1` | 日期前15(分/时)触发，08:00执行，每年重复 |
| 4 | `0` | `0` | `3` | `"08:00"` | `1` | 同第2次（重复保存）|
| 5 | `0` | `0` | `3` | `"08:00"` | `0` | 当天08:00触发，不重复 |

## 验证状态 ✅ 全部完成

| 参数 | 状态 | 来源 |
|------|------|------|
| `unit` 枚举（1/2/3） | ✅ 确认 | 浏览器抓包：unit=2+number=1 = 提前1小时 |
| `frequency` 枚举（0/1/2/3） | ✅ 确认 | 浏览器抓包：2=每月，3=每周 |
| `triggerId` 固定为 "2" | ✅ 确认 | 无 UI 选项，此触发类型固定值 |
